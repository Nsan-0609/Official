## 公開方法

1. GitHubで新規リポジトリを作成
2. `index.html`、`style.css`、`script.js`、`assets`フォルダ、`README.md`をアップロード
3. Settings → Pages を開く
4. Branch を `main`、Folder を `/root` に設定
5. Save
6. 数分後に公開URLが発行されます
